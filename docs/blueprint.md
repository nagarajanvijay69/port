# **App Name**: PORT

## Core Features:

- Secure Authentication & Profile: User signup and sign-in using phone number OTP verification, with options for setting a basic user profile including display name and avatar.
- Contact Management: Ability to import contacts from the phonebook (opt-in), add new contacts via QR code, and manage friend requests.
- 1:1 Real-time Text & Media Messaging: Send and receive instant text messages, images, and voice notes in one-on-one chats, including read receipts and typing indicators.
- Basic Group Messaging: Create and participate in small group conversations, allowing members to exchange messages and media.
- Moments Feed: Browse and create simple posts containing text and images, which are visible on a personal social feed to connected friends.
- Push Notifications & Presence: Receive real-time push notifications for new messages and friend requests, and view the online status ('last seen') of contacts.
- AI Reply Suggestions Tool: Utilize an AI tool to generate contextually relevant suggestions for message replies in 1:1 and group chats, or to assist with drafting social feed captions.

## Style Guidelines:

- Light color scheme evoking modernity and calm, inspired by the provided image's pastel gradients. Primary brand color: a vibrant mint green (#269160) for key interactive elements like buttons and branding. Background color: A very subtle, cool-toned off-white (#F7F6F8) that blends seamlessly with diverse UI elements. Accent color: A slightly brighter, complementary shade of green (#56C25E) to highlight secondary actions and achieve visual hierarchy.
- Font for both headlines and body text: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for clear communication in a social platform.
- Minimalist and clean line-art icons, consistent with the sleek and functional design shown in the provided screenshot (e.g., the phone icon in the input field).
- Responsive, mobile-first design with a clean, card-based interface for content presentation. Emphasis on generous whitespace and centered layouts, as exemplified by the sign-in modal in the reference image, to promote focus and ease of use.
- Subtle and fast micro-animations, particularly for transitions in messaging (e.g., sending/receiving messages, typing indicators) and content loading, to reinforce a perception of speed and reliability without causing distraction.
